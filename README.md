# UniqueRandom

Generate random non-repeating integers from a specified range in O(1) space.

## Features

### Complexity

- **O(1)** space
- **O(1)** time (amortized)

### Single file

UniqueRandom is single Java file, it is suggested to copy the source code into your project instead of adding a jar dependency.
